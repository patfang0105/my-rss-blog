# 📰 我的 RSS 聚合站

一个自动更新的 RSS 聚合博客，展示来自多个源的最新文章。

## 🌟 功能特性

- ✅ **自动更新**：每6小时自动更新内容
- ✅ **多源聚合**：整合多个 RSS 源
- ✅ **分类显示**：按主题分类文章
- ✅ **响应式设计**：支持手机和电脑访问
- ✅ **完全免费**：使用 GitHub Pages 托管

## 📊 RSS 源

- **Hacker News** - 技术新闻
- **CSIS** - 国际关系研究
- **CFR** - 国际关系分析和评论

## 🚀 访问地址

部署完成后，访问：`https://patfang0105.github.io/my-rss-blog`

## 🔧 技术栈

- **Jekyll** - 静态网站生成器
- **GitHub Pages** - 免费托管
- **GitHub Actions** - 自动更新
- **Python** - RSS 聚合脚本

## 📝 使用方法

1. 克隆仓库
2. 修改 `_config.yml` 中的 RSS 源
3. 运行 `python3 rss_aggregator.py` 生成文章
4. 推送到 GitHub 自动部署
